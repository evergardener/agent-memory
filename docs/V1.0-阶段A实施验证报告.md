# Agent Memory for Hermes — V1.0 阶段 A 实施验证报告

> 日期：2026-07-19<br>
> 版本：`1.0.0-rc.5`<br>
> 范围：主体身份、恒星/行星投影契约与历史噪声隔离

## 1. 结论

阶段 A 已完成并通过完整发布门禁。用户与 Hermes profile 人格现在由稳定 Subject 表示并投影为恒星；普通 canonical entity 投影为行星；同 profile 多实例不新增恒星；Subject 引用的实体不会重复成为行星；内部来源 ID、时间戳、自动化测试标签和句子片段不会成为天体。

本阶段没有实现关系社区、观察镜片、情节星座/长期星流的新投影，以及星系成员和布局治理。这些能力分别属于阶段 B/C/D，不应从本报告推定为已经完成。

## 2. 实施内容

### 2.1 身份与来源映射

- 新增 `core.subjects`、`core.subject_sources`，并为 `core.sources` 增加 `subject_id`；
- 每个 namespace 自动建立一个稳定的本地用户 Subject；
- Hermes source 默认按规范化 `source_profile` 解析 profile 人格 Subject，`source_instance`、session 和时间戳不参与身份键；
- 支持把来源人工映射到既有 Subject、撤销人工映射并回到自动规则；
- 支持人工修改 Subject 显示名和颜色；所有治理操作写入审计事件；
- 用户 Subject 不允许被隐藏，避免主宇宙失去第一人称中心。

### 2.2 星图投影

- 删除图 API 对 `core:user`、`core:hermes` 固定节点的依赖；
- 主宇宙以用户为中心，profile 人格按稳定双环位置排列；
- 子星系只显示该空间内容，不重复显示主体恒星群；
- Subject 详情展示类型、来源数量和可读信息，并提供名称/颜色编辑入口；
- 默认隐藏自动化测试 Subject，同时保留底层证据和审计；
- 普通实体节点继续使用唯一 canonical entity ID，事实和句子不进入天体集合。

### 2.3 噪声防线

- 实体抽取、确定性 worker、人工拆分和图投影共用内部标签阻断策略；
- `derived-*`、`agent-memory-*`、`relay-时间戳`、`isolated-时间戳` 等来源或自动化标签不再成为用户可见实体；
- 自动化测试空间和来源默认不污染主星图，但只读原始证据不被删除。

### 2.4 管理契约

- `GET /api/v1/graph/subjects`：读取 Subject 及脱敏来源映射；
- `PUT /api/v1/graph/subjects/{id}`：修改显示名、颜色或允许的显示状态；
- `POST /api/v1/graph/subjects/{id}/sources/{source_id}`：人工建立来源映射；
- `DELETE /api/v1/graph/subjects/{id}/sources/{source_id}`：撤销人工映射并恢复自动解析；
- 写操作要求 UI session，并保留治理审计。

## 5. 真实 staging 与浏览器验收

- 真实导入 staging `qishuo` profile 解析为一个 Hermes profile Subject；一个 source 精确映射，不因实例或会话增加恒星；
- staging 中 Subject 共 2 个：本地用户和 `qishuo` profile 人格；
- 旧 `core:*` 固定节点为 0，内部/自动化可见标签为 0，Subject/行星重复为 0；
- 默认主宇宙仅显示本地用户和真实 Hermes profile，不显示测试 profile；
- profile 恒星点击后可查看 Subject 详情并打开名称/颜色编辑弹窗；
- 进入子星系后 `.subject-cores` 不存在，主体恒星群不会重复绘制；
- 浏览器控制台 0 error。

## 6. 已知边界与下一阶段

建议下一步进入阶段 B，顺序为：

1. 把长期/阶段/当前从星系成员规则彻底拆为观察镜片；
2. 统一行星渲染与 canonical entity 详情，完成真实数据可读性抽检；
3. 定义有证据支撑的行星—行星关系输入，为阶段 C 社区发现做 Spike；
4. 在真实 staging 上先报告候选社区、桥接行星和深空比例，再选择社区算法及默认阈值。

阶段 B 不应直接锁定社区算法。当前真实数据关系密度可能不足，先用可解释指标验证输入质量，可以避免为了视觉效果制造虚假星系。
